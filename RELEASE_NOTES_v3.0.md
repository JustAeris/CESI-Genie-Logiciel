# EasySave v3.0 — Release Notes

## Nouveautés

### Parallélisme
- Les sauvegardes s'exécutent maintenant en parallèle

### Contrôle des travaux
- Pause / Resume / Stop par travail en temps réel

### Logiciel métier
- Pause automatique si un logiciel métier est détecté

### CryptoSoft
- Un seul chiffrement à la fois via SemaphoreSlim

### Logs centralisés
- Envoi des logs vers un serveur Docker distant

### Fichiers prioritaires
- Extensions prioritaires transférées en premier

Mai 2026
